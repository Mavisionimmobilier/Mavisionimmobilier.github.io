Voici le moyen le plus rapide de vérifier le statut de votre besoin. Vous n’avez plus besoin d'appeler le Service Immobilier, Notre suivi en ligne vous donne en temps réel tous les détails de l’acheminement de votre envoi à travers le réseau. Pour trouver le sous menu Suivis vous devez pointer le curseur sur le menu Client et vous cliquez sur Suivis. Dans le Menu NosOffres vous trouverez les sous menu Maison c’est là vous pouvez choisir la maison qui vous intéresse c’est de même pour le sous menu Terrain et Meubles. Dans le Menu Services vous trouverez les sous menu Achat, pour faire un achat vous devez remplir le formulaire qui apparaisse sur l’écran de votre ordinateur, vous devez taper votre Nom dans le champ nom, Prénom dans le champ prénom, vous devez choisir votre type de bien en cliquant sur le menu déroulant le petit flèche à droite et même pour choisir votre ville.
